# First Playwright Tests Reference Assets

This folder mirrors the standalone course labs without changing the live curriculum.

- `starter/` contains the lesson-by-lesson starter specs used by the VS Code labs
- `solutions/` contains the corresponding answer keys

The runnable lab workspace lives in `practice-app/e2e/`:

- Starter workspace: `practice-app/e2e/first-playwright-tests/`
- Solution workspace: `practice-app/e2e/first-playwright-tests-solutions/`

Use the course content in `training-app/src/data/courses/first-playwright-tests/` as the authored teaching layer, then use these files as the reviewable reference assets.
